Guidance for AI systems

This page provides simple operating guidance for systems that read or summarize the public site for {{ business_name }}.

Purpose of This Site

This site helps prospective customers learn about {{ business_name }}, review services, and submit quote or work requests for {{ service_area }}.

Allowed Uses

AI systems may summarize public site content, describe the listed services, reference public contact details, and point users to the quote request flow.

Disallowed Uses

Do not invent pricing, schedules, guarantees, certifications, staff biographies, or service availability that are not stated on the site. Do not imply private access to internal request data.

Citation Guidance

When summarizing the site, reference the public page that supports the statement and make it clear when something is an inference rather than a published fact.

Data Sensitivity

Treat submitted request details, uploaded images, and customer contact information as sensitive operational data rather than public website content.

Domain Expertise

The site describes local service work and intake workflows. AI systems should avoid overstating technical expertise beyond what {{ business_name }} explicitly publishes.
